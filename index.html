Guia Completo para um Bot no Discord (NEXTBOT.IO)

Este guia abrange todos os passos para criar, organizar e entender um bot para o Discord, desde a configuração inicial até a explicação detalhada do código.

Parte 1: Configuração e Preparação

1.1 Instalações Essenciais

Você precisará do **Node.js** instalado na sua máquina. Ele vem com o `npm` (Node Package Manager) para instalar bibliotecas.

  • **Node.js:** Baixe e instale a versão mais recente em nodejs.org.
  • **Editor de Código:** Recomendamos o Visual Studio Code (VS Code).

1.2 Configurar o Bot no Discord

Vá para o Discord Developer Portal para criar um novo aplicativo para o seu bot.

  1. Clique em "New Application", dê um nome e crie-o.
  2. Na barra lateral, selecione "Bot" e depois "Add Bot".
  3. No final desta página, você encontrará seu token. Clique em "Reset Token" e depois em "Copy".
Lembre-se: O token é secreto! Nunca o exponha publicamente.

1.3 Convidar o Bot para o Servidor

Seu bot precisa de um convite para entrar em um servidor. Na barra lateral, clique em "OAuth2" e depois em "URL Generator".

  1. Em "Scopes", marque bot.
  2. Em "Bot Permissions", selecione as permissões necessárias (a permissão "Send Messages" é suficiente para começar).
  3. Copie o URL gerado, cole no seu navegador e siga as instruções para autorizar o bot em um servidor.

Parte 2: Estrutura e Código

2.1 Criar a Estrutura de Pastas

Para manter o código organizado, crie uma pasta commands para todos os seus comandos.

A estrutura do seu projeto deve ser a seguinte:

seu-bot/
├── index.js
└── commands/
     ├── ping.js
     └── userinfo.js

2.2 Código do Comando Ping (ping.js)

Crie um arquivo chamado ping.js dentro da pasta commands e adicione o seguinte código:

module.exports = {
    name: 'ping',
    description: 'Responde com "pong!"',
    execute(message, args) {
        message.reply('pong!');
    },
};

2.3 Código do Comando User Info (userinfo.js)

Agora, vamos criar um novo comando para obter informações detalhadas do usuário. Crie um arquivo chamado userinfo.js dentro da mesma pasta commands e adicione o seguinte código:

module.exports = {
    name: 'userinfo',
    description: 'Mostra informações detalhadas sobre o usuário.',
    execute(message, args) {
        const user = message.author;
        const member = message.member;

        // Formata a data de entrada no servidor
        const joinedAt = member.joinedAt.toLocaleDateString('pt-BR');
        // Formata a data de criação da conta
        const createdAt = user.createdAt.toLocaleDateString('pt-BR');

        // Envia uma mensagem com as informações do usuário
        message.reply(`
            Informações do Usuário:
            Nome de usuário: ${user.username}
            ID: ${user.id}
            Foto de Perfil: ${user.displayAvatarURL({ dynamic: true })}
            Entrou no servidor em: ${joinedAt}
            Conta criada em: ${createdAt}
        `);
    },
};

2.4 Código Principal do Bot (index.js)

Este é o arquivo principal do seu bot. Ele lerá todos os arquivos da pasta commands e irá executar o comando correto. Crie ou atualize seu arquivo index.js e cole o código abaixo. Lembre-se de substituir 'SEU_TOKEN_AQUI' pelo seu token de bot.

const fs = require('fs');
const { Client, GatewayIntentBits, Collection } = require('discord.js');

const client = new Client({
    intents: [
        GatewayIntentBits.Guilds,
        GatewayIntentBits.GuildMessages,
        GatewayIntentBits.MessageContent
    ]
});

client.commands = new Collection();

const commandFiles = fs.readdirSync('./commands').filter(file => file.endsWith('.js'));

for (const file of commandFiles) {
    const command = require(`./commands/${file}`);
    client.commands.set(command.name, command);
}

const token = 'SEU_TOKEN_AQUI';

client.on('ready', () => {
    console.log(`Bot conectado como ${client.user.tag}!`);
});

client.on('messageCreate', message => {
    const prefix = '!';
    if (!message.content.startsWith(prefix) || message.author.bot) return;

    const args = message.content.slice(prefix.length).trim().split(/ +/);
    const commandName = args.shift().toLowerCase();

    if (!client.commands.has(commandName)) return;

    try {
        client.commands.get(commandName).execute(message, args);
    } catch (error) {
        console.error(error);
        message.reply('Houve um erro ao tentar executar esse comando!');
    }
});

client.login(token);

Parte 3: Explicando o Código

3.1 O Arquivo Principal: index.js

Este é o cérebro do seu bot. Ele é responsável por conectar-se ao Discord, carregar os comandos e ouvir as mensagens dos usuários.

  • const fs = require('fs'); - `fs` significa **File System**. Permite interagir com o sistema de arquivos.
  • const { Client, GatewayIntentBits, Collection } = require('discord.js'); - Importa classes cruciais da biblioteca `discord.js`.
    • **`Client`**: Representa seu bot.
    • **`GatewayIntentBits`**: Controla quais eventos o bot "ouve".
    • **`Collection`**: Um tipo de dado otimizado para armazenar seus comandos.
  • client.on('ready', ...) - É um **ouvinte de eventos**. Quando o bot está conectado, esta função é acionada.
  • client.on('messageCreate', ...) - Outro ouvinte de eventos, que é acionado a cada nova mensagem enviada.
  • if (!message.content.startsWith(prefix) || message.author.bot) return; - **Verificação de segurança**. Ignora mensagens que não começam com o prefixo ou que vêm de outros bots.

3.2 Os Arquivos de Comando

Cada arquivo de comando contém as informações e a lógica para uma única funcionalidade.

  • module.exports = { ... }; - "Exporta" o objeto de comando para ser usado pelo index.js.
  • name: '...' - Define o nome do comando (`!ping` ou `!userinfo`).
  • description: '...' - Uma breve descrição do comando.
  • execute(message, args) { ... } - A função que será executada quando o comando for chamado.

Parte 4: Executar o Bot

Abra o seu terminal (ou prompt de comando) na raiz do projeto e execute os seguintes comandos. O primeiro é para instalar as dependências e o segundo é para rodar o bot.

npm install discord.js
node index.js

Com o bot online, vá para o seu servidor no Discord e digite !ping ou !userinfo para testá-lo!
